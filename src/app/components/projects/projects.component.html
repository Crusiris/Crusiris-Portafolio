C-Task

C-Task: Proyecto web que permite que un usuario se Registra e inicie sesion, ademas de manejar sus proyectos y organizar tareas de dichos proyectos, El FrontEnd de esta aplicación fue creado con React, Reducer, Hooks, State, y el BackEnd fue desarrollado con Node js, Mongodb y Express.
